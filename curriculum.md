# ITR 2026-2027 Internship Program - Curriculum

## 1. Introduction

| Topic | Details |
|-------|---------|
| **Program Overview** | Comprehensive internship program covering full-stack web development and professional skills |
| **Objectives** | Equip interns with practical knowledge in web development, version control, and project management |
| **Learning Outcomes** | Master Python, web technologies, backend frameworks, and collaborative development practices |

---

## 2. Python Programming

| Module | Topics |
|--------|--------|
| **Fundamentals of Python** | Variables, Data Types, Control Flow, Functions, Modules |
| **Data Structures** | Lists, Tuples, Dictionaries, Sets, Collections, Sorting & Filtering |
| **Object-Oriented Programming** | function, Classes, Objects |
| **File Handling & Exception Management** | File I/O Operations, Error Handling, Custom Exceptions |

---

## 3. HTML & CSS

| Module | Topics |
|--------|--------|
| **Basics of HTML** | Document Structure, Semantic Tags, Forms, Input Elements, Accessibility |
| **Styling with CSS** | Selectors, Box Model, Flexbox, Grid, Responsive Design, Media Queries, Animations |

---

## 4. JavaScript

| Module | Topics |
|--------|--------|
| **Introduction to JavaScript** | Syntax, Variables, Data Types, Operators, Control Flow, Functions |
| **DOM Manipulation** | Selecting Elements, Creating & Modifying Content, Traversing the DOM |
| **Event Handling** | Event Listeners, Common Events |

---

## 5. Python Backend Development

| Module | Topics |
|--------|--------|
| **Web Frameworks (Flask)** | Routing, Request/Response Handling, Templates |
| **RESTful APIs** | API Design Principles, HTTP Methods, Status Codes, JSON Handling |
| **Database Integration** | SQL Basics, ORM (SQLAlchemy), Database Design, Migrations, Optimization |

---

## 6. Version Control with Git

| Module | Topics |
|--------|--------|
| **Git Basics** | Repository Creation, Commits, History, Staging, Status, Diff |
| **Branching and Merging** | Creating Branches, Branch Management, Merging Strategies, Conflict Resolution |
| **Collaboration on GitHub** | Remote Repositories, Push/Pull, Pull Requests, Code Review, Issues & Discussions |

---

## 7. Project Work

| Phase | Activities |
|-------|-----------|
| **Group Projects** | Team Formation, Project Planning, Task Distribution, Sprint Planning |
| **Development & Implementation** | Coding, Testing, Documentation, Version Control, Peer Review |
| **Reporting & Documentation** | Technical Documentation, README Files, Code Comments, Architecture Diagrams |
| **Final Presentation & Evaluation** | Project Showcase, Live Demo, Q&A Session, Performance Evaluation, Feedback |

---

## Summary

This comprehensive curriculum provides a structured learning path from fundamental programming concepts through advanced backend development and professional collaboration practices. Interns will gain hands-on experience with industry-standard tools and best practices throughout the program.
